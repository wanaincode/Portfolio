# Portfolio

個人ポートフォリオサイトとして、自己紹介、研究内容、プロジェクト、履歴書をまとめている。

## 📂 フォルダ構成
```text
portfolio/
├─ README.md               ⬅️ このファイル
├─ index.html              ⬅️ トップページ
├─ resume.html             ⬅️ 履歴書ページ
├─ research.html           ⬅️ 研究紹介ページ
├─ projects.html           ⬅️ プロジェクト紹介ページ
├─ projects-data.js        ⬅️ プロジェクトデータ（JSON形式）
└─ images/                 ⬅️ 使用画像
```
